# react-native-alarm-clock

React Native Alarm Clock

## Installation

```sh
npm install react-native-alarm-clock
```

## Usage

```js
import AlarmClock from "react-native-alarm-clock";

// ...

const result = await AlarmClock.multiply(3, 7);
```
